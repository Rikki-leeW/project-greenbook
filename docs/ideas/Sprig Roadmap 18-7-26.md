# 🌿 Sprig Roadmap

> "Every garden grows one careful season at a time."
>
> This roadmap guides the development of Sprig.
> We build slowly, thoughtfully, and never rush ahead.
> Every new feature should feel as though it belongs in an old gardener's notebook.
>
> Before adding anything new, always ask:
>
> **"Would another gardening app have done this?"**
>
> If the answer is yes, keep thinking.

# 🌳 Development Principles

Always build the roots before the branches.

One roadmap item at a time.

Commit completed work before starting the next feature.

Never sacrifice beauty for speed.

Prefer timeless over trendy.

Prefer stories over statistics.

Prefer reflection over achievement.

Every feature should help gardeners build their own growing wisdom.

If another gardening app would build it the same way...

...keep thinking.

Every page should feel like opening an old gardener's notebook.
